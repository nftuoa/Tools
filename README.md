## 使用说明
1. 登录[中小学智慧教育平台](https://www.zxx.edu.cn/tchMaterial)
2. 选择需要下载的课本
3. 在浏览器地址栏找到contentid=******^%$^$^%&*&处，并将contentid复制下来
4. 运行程序，输入contentid，回车后即可下载。
